```cpp
move_iterator operator-(difference_type n) const;           // C++11
constexpr move_iterator operator-(difference_type n) const; // C++17
```

## 概要
イテレータを`n`回逆に進める。


## 戻り値
`return move_iterator(`[`base`](base.md)`() - n);`


## 例
```cpp example
#include <iostream>
#include <vector>
#include <memory>
#include <iterator>

int main()
{
  std::vector<std::unique_ptr<int>> v;
  for (int i = 0; i < 5; ++i)
    v.emplace_back(new int(i));

  auto it = std::make_move_iterator(v.end());
  auto it2 = it - 2;
  std::unique_ptr<int> p = *it2;

  std::cout << *p << std::endl;
}
```
* it - 2[color ff0000]
* v.emplace_back[link /reference/vector/vector/emplace_back.md]
* std::make_move_iterator[link /reference/iterator/make_move_iterator.md]
