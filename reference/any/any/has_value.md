```cpp
bool has_value() const noexcept;
```

## 概要
有効な値を保持しているかを判定する。


## 効果
有効値を保持している場合に`true`を返し、そうでなければ`false`を返す。


## 例
```cpp example
#include <cassert>
#include <any>

int main()
{
  std::any x = 3;
  assert(x.has_value() == true);

  std::any y;
  assert(y.has_value() == false);
}
```
