## 概要
末尾の要素を指す逆順イテレータを取得する。


## 戻り値
```cpp
return reverse_iterator(cend());
```
* cend()[link cend.md]


## 例
```cpp example
#include <iostream>
#include <stacktrace>
#include <algorithm>

void g() {
  std::stacktrace st = std::stacktrace::current();
  std::for_each(st.rbegin(), st.rend(), [](const std::stacktrace_entry& x) {
    std::cout << x << std::endl;
  }):
}

void f() {
  g();
}

int main() {
  f();
}
```
