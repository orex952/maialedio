## 概要
スタックトレースの履歴数が空かどうかを判定する。


## 戻り値
保持しているスタックトレースの履歴が空であれば`true`、そうでなければ`false`を返す。


## 例
```cpp example
#include <cassert>
#include <stacktrace>

void g() {
  auto trace = std::stacktrace::current(0, 0);
  assert(trace.empty());
}

void f() {
  g();
}

int main() {
  f();

  std::stacktrace trace{};
  assert(trace.empty());
}
```
