```cpp
virtual ~pool_resource();
```

## 概要
管理しているすべてのメモリを解放する。

クラス名を`pool_resource`としているのは説明のためのプレースホルダで、`synchronized_pool_resource`と`unsynchronized_pool_resource`で共通ということである。

## 効果

`this->release()`を呼び出す。
