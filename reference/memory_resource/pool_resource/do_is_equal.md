bool do_is_equal(const memory_resource& other) const noexcept override;

## 概要
ある`memory_resource`オブジェクトが`*this`自身であるかを調べる。

## 引数
`other` -- 任意の`memory_resource`オブジェクト

## 戻り値
`return this == &other;`

## 例外
投げない。
