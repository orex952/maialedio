## 概要
`year`オブジェクトを`int`型に明示的に変換する。


## 戻り値
- (1) : メンバ変数として保持している`int`型としての年の値を返す


## 例
```cpp example
#include <cassert>
#include <chrono>

namespace chrono = std::chrono;

int main()
{
  chrono::year y{2020};
  auto value = static_cast<int>(y);
  assert(value == 2020);
}
```
