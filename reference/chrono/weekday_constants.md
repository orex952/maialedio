## 概要
曜日を表す定数。

- (1) : 日曜日を表す定数
- (2) : 月曜日を表す定数
- (3) : 火曜日を表す定数
- (4) : 水曜日を表す定数
- (5) : 木曜日を表す定数
- (6) : 金曜日を表す定数
- (7) : 土曜日を表す定数


## 例
### 基本的な使い方
```cpp example
#include <cassert>
#include <iostream>
#include <chrono>

namespace chrono = std::chrono;

int main() {
  chrono::weekday w = chrono::Sunday;
  ++w;
  std::cout << w << std::endl;

  chrono::weekday v = chrono::Wednesday;
  v += chrono::days{3};
  std::cout << v << std::endl;

  // 曜日は循環する
  chrono::weekday u = chrono::Saturday;
  v += chrono::days{2};
  assert(v == chrono::Monday);
}
```
