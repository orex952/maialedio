## 概要
インデックス付き曜日要素を取得する。


## 戻り値
コンストラクタで設定されて保持しているインデックス付き曜日オブジェクトを返す。


## 例
```cpp example
#include <cassert>
#include <chrono>

namespace chrono = std::chrono;

int main()
{
  chrono::month_weekday mwd = chrono::March/chrono::Sunday[1];

  chrono::weekday_indexed wdi = mwd.weekday_indexed();
  assert(wdi == chrono::Sunday[1]);
}
```
