```cpp
namespace std::chrono {
  constexpr bool
    operator==(const year_month_weekday& x,
               const year_month_weekday& y) noexcept; // (1) C++20
}
```

## 概要
`year_month_weekday`同士の等値比較を行う。


## 戻り値
- (1) :

```cpp
return x.year() == y.year() &&
       x.month() == y.month() &&
       x.weekday_indexed() == y.weekday_indexed();
```

```cpp example
}
```
