## 例
### 基本的な使い方
```cpp example
#include <iostream>
#include <chrono>

namespace chrono = std::chrono;

int main() {
  chrono::month m = chrono::January;
  ++m;
  std::cout << m << std::endl;

  chrono::month n = chrono::March;
  n += chrono::months{3};
  std::cout << n << std::endl;
}
```
