```cpp
namespace std::chrono {
  constexpr bool operator==(const month_day& x, const month_day& y) noexcept; // (1) C++20
}
```

## 概要
`month_day`同士の等値比較を行う。


## 戻り値
- (1) :

```cpp
return x.month() == y.month() && x.day() == y.day();
```
