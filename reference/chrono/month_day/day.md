## 概要
日要素を取得する。


## 戻り値
コンストラクタで設定されて保持している日オブジェクトを返す。


## 例
```cpp example
#include <cassert>
#include <chrono>

namespace chrono = std::chrono;
using namespace std::chrono_literals;

int main()
{
  chrono::month_day md = chrono::March/1;

  chrono::day d = md.day();
  assert(d == 1d);
}
```
