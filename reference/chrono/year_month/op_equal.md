```cpp
namespace std::chrono {
  constexpr bool operator==(const year_month& x, const year_month& y) noexcept; // (1) C++20
}
```

## 概要
`year_month`同士の等値比較を行う。


## 戻り値
- (1) :

```cpp
return x.year() == y.year() && x.month() == y.month();
```
