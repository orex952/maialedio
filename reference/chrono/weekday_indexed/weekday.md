## 概要
曜日を取得する。


## 戻り値
コンストラクタで設定されて保持している[`weekday`](/reference/chrono/weekday.md)型オブジェクトを返す。


## 例
```cpp example
#include <cassert>
#include <chrono>

namespace chrono = std::chrono;

int main()
{
  chrono::weekday_indexed wi = chrono::Sunday[1];
  chrono::weekday w = wi.weekday();

  assert(w == chrono::Sunday);
}
```
