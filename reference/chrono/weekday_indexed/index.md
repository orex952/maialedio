## 概要
インデックスを取得する。


## 戻り値
コンストラクタで設定されて保持している曜日のインデックスを返す。


## 例
```cpp example
#include <cassert>
#include <chrono>

namespace chrono = std::chrono;

int main()
{
  chrono::weekday_indexed wi = chrono::Sunday[1];
  unsigned int index = wi.index();

  assert(index == 1);
}
```
