# execution

* cpp17[meta cpp]

`<execution>`ヘッダでは、アルゴリズムの並列実行を許可するための実行ポリシー型を定義する。


| 名前 | 説明 | 対応バージョン |
|------|------|----------------|
| [`is_execution_policy`](execution/is_execution_policy.md) | 型が実行ポリシーかを判定する | C++17 |
| [`execution`](execution/execution.md)                     | 実行ポリシー用の名前空間 | C++17 |
