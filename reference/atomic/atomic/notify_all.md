## 概要
待機している全てのスレッドを起床させる。

この関数は、[`wait()`](wait.md)関数によるブロッキング待機を解除する。


## テンプレートパラメータ制約
- (1) :
    - C++20 : `atomic<T>::is_always_lock_free`が`true`であること


## 効果
起床待機している全てのアトミックオブジェクトの待機を解除する


## 戻り値
なし


## 例外
投げない


## 例
```cpp example
#include <iostream>
#include <atomic>
#include <thread>

int main()
{
  std::atomic<bool> x {false};

  auto f = [&x] {
    while (true) {
      x.wait(false);
      if (x.load() == true) {
        break;
      }
    }
  };

  std::thread t1 {f};
  std::thread t2 {f};

  x.store(true);
  x.notify_all();

  t1.join();
  t2.join();
}
```
