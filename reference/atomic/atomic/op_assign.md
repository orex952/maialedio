```cpp
atomic& operator=(const atomic&) = delete;          // (1) C++11

```

## 概要

## テンプレートパラメータ制約
- (3) :
    - C++20 : `atomic<T>::is_always_lock_free`が`true`であること


## 効果

## 戻り値
`desired`


## 例外

## 例
```cpp example
#include <iostream>
#include <atomic>

int main()
{
  std::atomic<int> x(1);

  x = 2;

  std::cout << x.load() << std::endl;
}
```
