# operator T
* atomic[meta header]
* std[meta namespace]

```cpp
operator T() const volatile noexcept; // (1) C++11
operator T() const noexcept;          // (2) C++11
```

## 概要
型`T`への暗黙の型変換


## テンプレートパラメータ制約
- (1) :
    - C++20 : `atomic<T>::is_always_lock_free`が`true`であること


## 戻り値
[`load()`](load.md)


## 例外
投げない


## 例
```cpp example
#include <iostream>
#include <atomic>

int main()
{
  std::atomic<int> x(1);

  int value = x;
  std::cout << value << std::endl;
}
```

```
1
```
