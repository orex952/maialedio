## 概要
`stop_source`オブジェクトの非等値比較を行う。

## 戻り値
`!(x == y)`

## 例外
投げない。

## 例
```cpp example
#include <cassert>
#include <stop_token>

int main()
{
  std::stop_source ss1;
  std::stop_source ss2 = ss1;
  std::stop_source ss3;
  std::stop_source ss4(std::nostopstate);
  std::stop_source ss5(std::nostopstate);;

  assert(ss1 == ss2);
  assert(ss1 != ss3);
  assert(ss1 != ss4);
  assert(ss4 == ss5);
}
```
