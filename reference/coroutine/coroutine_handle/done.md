```cpp
bool done() const;
```

## 概要
最終サスペンドポイントで中断状態にあること、つまりコルーチン本体の終端到達済みか確認する。


## 事前条件
`*this`が中断状態のコルーチンを指すこと。


## 戻り値
コルーチンが最終サスペンドポイントで中断状態にあれば`true`を返す。
それ以外の箇所で中断状態にあれば`false`を返す。
