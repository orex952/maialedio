```cpp
coroutine_handle& operator=(nullptr_t) noexcept;
```

## 概要
コルーチンハンドルをリセットし、有効なコルーチンを指さない状態にする。


## 戻り値
`*this`


## 事後条件
[`address()`](address.md) `== nullptr`


## 例外
投げない


## 例
```cpp example
#include <cassert>
#include <coroutine>

int main()
{
  std::coroutine_handle<> h;
  h = nullptr;
  assert(h.address() == nullptr);
}
```
