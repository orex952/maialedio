# swap
* list[meta header]
* std[meta namespace]
* list[meta class]
* function[meta id-type]

```cpp
void swap(list& x);
void swap(list& x) noexcept(allocator_traits<Allocator>::is_always_equal::value); // C++17
```

## 概要
他の`list`オブジェクトと値を入れ替える。


## 効果
`*this`の内容を`x`と交換する。


## 戻り値
なし


## 計算量
定数時間


## 例
```cpp example
#include <iostream>
#include <list>

int main()
{
  std::list<int> ls1 = {1, 2, 3};
  std::list<int> ls2 = {4, 5, 6};

  ls1.swap(ls2);

  for (int x : ls1) {
    std::cout << x << std::endl;
  }

  std::cout << std::endl;

  for (int x : ls2) {
    std::cout << x << std::endl;
  }
}
```
